Bootstrap – rapid UI framework

Bootstrap is the world's most popular CSS framework. It provides pre‑built components, a responsive grid system, and utility classes to build modern websites quickly without writing custom CSS from scratch. This very page is styled with Bootstrap!

⚖️ Bootstrap vs writing custom CSS

AspectCustom CSSBootstrap (what's better)
Development speed Slow – you write everything from scratch 🚀 Fast – prebuilt components and utilities speed up prototyping
Responsive design You must implement media queries yourself 📱 Built‑in responsive grid and breakpoints – just use classes like col-md-4
Consistency Easy to have inconsistent spacing, colors, typography 🎨 Design system with standardized variables, themes, and utilities ensures consistency
Browser compatibility You handle cross‑browser issues manually 🌍 Bootstrap is tested across all modern browsers – saves time
Customization Total control, but more work ⚙️ Easy to override with custom CSS or Sass variables – best of both worlds

Bootstrap is ideal for rapid development and learning; it doesn't prevent you from writing custom CSS when needed.
