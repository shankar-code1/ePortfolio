📦 The box model

Every element is a box: margin (outside), border, padding (inside), and content.

🧩 Flexbox – one‑dimensional layout

Flexbox makes alignment and distribution easy. This container uses display: flex with gap and justify-content.

1
2
3
.flex-container {
  display: flex;
  gap: 1rem;
  justify-content: center;
}
.flex-item {
  background: #1b4a6b;
  color: white;
  padding: 0.8rem 1.5rem;
  border-radius: 2rem;
}

📊 CSS3 improvements

FeatureOlder CSSCSS3 (what's better)
Selectors Basic selectors (class, ID, type) Attribute selectors, :nth-child(), :not(), ::before/::after – more precise targeting.
Borders & backgrounds Solid borders, single background image Border-radius, multiple backgrounds, gradients, border-image.
Effects No built‑in shadows, transforms, or animations box-shadow, text-shadow, transform (scale, rotate), transition, @keyframes.
Layout Floats, positioning, tables Flexbox, Grid – powerful, responsive layouts with less code.
Responsive design No media queries @media queries adapt styles to screen size, making sites mobile‑friendly.

CSS3 turns static pages into modern, interactive experiences.
