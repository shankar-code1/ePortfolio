JavaScript – making pages interactive

JavaScript is the programming language of the web. It allows you to create dynamic content, control multimedia, animate images, and handle user interactions. Everything from form validation to complex single-page applications is powered by JS.

📘 Core JavaScript concepts

ConceptDescriptionCode example
Variables Store data values. let and const are block‑scoped; var is function‑scoped. let count = 0; const MAX = 50;
Functions Reusable blocks of code. Can be declarations, expressions, or arrow functions. function add(a,b) { return a+b; }
const square = x => x*x;
Events React to user actions like clicks, typing, or mouse movements. element.addEventListener('click', () => { ... });
DOM manipulation Access and modify HTML elements dynamically. document.getElementById('id').innerText = 'Hello';
Arrays Ordered lists of items. Many built‑in methods like push, map, filter. let fruits = ['apple','banana'];
fruits.push('orange');
Objects Key‑value pairs for structured data. let user = { name: 'Shankar', age: 24 };
Template literals Strings with embedded expressions using backticks. `Hello, ${user.name}!`

⚡ ES5 vs ES6+ (modern JavaScript)

FeatureES5 (older)ES6+ (modern)
Variable declaration var (function‑scoped, can be redeclared) let & const (block‑scoped, no redeclaration)
Functions function keyword only Arrow functions () => {} – shorter and lexical this
String concatenation 'Hello ' + name Template literals `Hello ${name}`
Array iteration for loop, forEach (ES5 added) map, filter, reduce, for...of
Objects Basic object literals Destructuring, spread operator {...obj}, shorthand properties
Modules No native module system (used script tags or CommonJS) import / export – native modules
Promises No native promises (used callbacks or libraries) Native Promise, async/await for clean asynchronous code

Modern JavaScript (ES6+) makes code cleaner, safer, and more expressive.

⏳ Asynchronous JS – callbacks, promises, async/await

JavaScript handles operations that take time (like fetching data from an API) without freezing the page. Here's a quick example of fetching fake user data.

Click the button to load data. 
async function fetchUser() {
  const res = await fetch('https://randomuser.me/api/');
  const data = await res.json();
  const user = data.results[0];
  return `${user.name.first} ${user.name.last}`;
}
